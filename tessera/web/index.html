Import data

Items need a text column (CSV) or field (JSONL); id is optional. A new dataset also needs a rubric — start from the template below. Re-importing the same ids updates them (idempotent).

rubric template 
{
  "id": "my-rubric", "name": "My labels", "labels": ["a", "b"],
  "definitions": {"a": "when …", "b": "when …"},
  "guidelines": "Pick the label a careful teammate would."
}

Rubric

The model is prompted with exactly this text. When labels disagree with how YOUR team labels, fix the words here — measured, one rewritten paragraph moved coverage 11%→91% on a real corpus.

Gold labels

Gold is the ground truth the gate calibrates against — the precision promise is only as good as this sample. Tessera picks items spread across the corpus; you label them with the keyboard. 50–150 is the useful range.
